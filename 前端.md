## Vue渲染原理
+ 和React是类似的
+ 维护虚拟dom，检测dom的变化
+ 当state改变，通过diff算法检测出dom的变化，对变化部分的dom进行重新渲染

## Koa的插件式架构
+ 和golang的Gin是类似的
+ 插件(middleware)是一层一层的
+ 请求先进入最外层、次外层、、、内层
+ 在任一层都可以直接返回请求而避免进入更深的层次
+ 返回时内层、次内层、、、最外层
+ 伪代码
```
def plugin(ctx):
    pre_handle(ctx)
    next_plugin(ctx)
    post_handle(ctx)
    return 
```
