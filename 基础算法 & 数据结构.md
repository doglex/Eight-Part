## 常见刷题用算法
+ 二分法/分治法： 题目一般会提示有序
+ HashMap(dict)： 提供O(1)缓存，可以给递归(自顶向下)用
+ DP: 自底向上的进行缓存，可以给for循环用
+ DFS(即、回溯)： 树的访问、网格的访问，一般用递归来写。val访问顺序决定是前、中、后序遍历
```python 
cache = {}
dfs(node, args):
    if not node:
        return output 
    dfs(node.left)
    cache[node] = node.value // 该行是前、中、后
    dfs(node.right)
    return output 
```
